VAP du Capital au dcs avec prime annuelle

Vap_capannu est la fonction R permettant de calculer la VAP du Capital au dcs avec prime annuelle partir dune table de mortalit.

Pour la fonction Vkx.capannu(x, k, K) :

x : Lge de lassur au moment du contrat. k : Le nombre dannes coules depuis le dbut du contrat jusqu la date de valorisation ( laquelle vous souhaitez calculer la VAP). K : Le capital dcs pour lequel les bnficiaires recevront un montant en euros.

Vkx.capannu <- function(x, k, K)
{
  prem <- Axn(TD2016tb, x=x)*K / axn(TD2016tb, x=x, m=0)
  K * Axn(TD2016tb, x=x+k) - prem * axn(TD2016tb, x=x+k, m=0)
}

VAP du Contrat de retraite

Vkx.retraite est la fonction R permettant de calculer la VAP du Contrat de retraite partir dune table de mortalit.

Pour la fonction Vkx.retraite(x, n, k, R) :

x : Lge de lassur au moment du contrat. n : Lge de dpart la retraite (par exemple, 65 ans). k : Le nombre dannes coules depuis le dbut du contrat jusqu la date de valorisation. R : Le montant de la rente en euros que lassur recevra partir de lge de dpart la retraite.

Vkx.retraite <- function(x, n, k, R)
{ 
  prem <- axn(TD2016tb, x=x, m=n-x)*R / axn(TD2016tb, x=x, m=0, n=n-x)
  k1 <- k[k < n-x]
  k2 <- k[k >= n-x]
  res <- axn(TD2016tb, x=x+k, m=n-x-k)*R - prem * axn(TD2016tb, x=x+k, m=0, n=n-x-k)
  c(res, axn(TD2016tb, x=x+k)*R)
}
